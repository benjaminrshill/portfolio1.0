brownspunbear.github.io/imaje

Imaje seeks to give artists creative control over their photo or painting gallery using a combination of the HTML5 drag & drop API and React. Functionality includes being able to add and remove images, add and remove rows, rearrange images, move rows up and down, change spacing between images, and change the size of the gallery itself.

The name Imaje may or may not stand for "I made a JavaScript exhibition".

bluesungfruit.com

A showcase for my personal artwork, the gallery was my first foray into JavaScript.

As well as the usual gallery features (e.g. click an image to enlarge, then move through one by one), each gallery row resizes based on the aspect ratios of the images inside, so that any number of images, of any width-height ratio, can be added to fit neatly into the row.
